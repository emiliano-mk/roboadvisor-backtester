# RoboAdvisor Backtester

Un proyecto de asesor financiero automatizado con backtesting y optimización de estrategias.

## Estructura del proyecto
- `app/core/`: contiene la lógica de datos, estrategias y simulación.
- `app/utils/`: utilidades como logs, configuración y visualización.
- `tests/`: pruebas unitarias.
- `notebooks/`: experimentos en Jupyter.

## Cómo ejecutar
```bash
source .venv/Scripts/activate
python app/main.py
